# QA任务进度提醒工具

基于 Python PyQt5 的 Windows 桌面应用程序

## 功能特性

### 界面设计
* **应用名**：QA任务进度提醒工具
* **左侧面板**：任务配置区域，包含文件选择和推送控制
* **右侧面板**：实时时钟显示 + 模拟键盘输入功能

### 核心功能

1. **任务文件选择**：支持 JSON 或纯文本格式的任务配置文件
2. **定时推送**：自动调用飞书 webhook 进行消息推送（每60秒一次）
3. **模拟键盘输入**：通过 pynput 库实现真实键盘输入模拟

## 安装依赖

```bash
pip install -r requirements.txt
```

或单独安装：
```bash
pip install PyQt5 pynput requests
```

## 运行程序

```bash
python main.py
```

## 任务文件格式

### JSON格式（推荐）
```json
[
    {"task": "任务名称", "priority": "高", "status": "进行中"},
    {"task": "另一个任务", "priority": "中", "status": "待开始"}
]
```

### 纯文本格式
每行一个任务名称，系统会自动设置默认优先级和状态。

## 关于"云枢监控"兼容性说明

**模拟键盘输入功能分析**：

本程序使用 `pynput` 库实现键盘输入模拟，其工作原理是：
- 通过系统底层 API 发送真实的键盘事件
- 这些事件与物理键盘按键在系统层面是等效的
- 大多数行为监控软件（包括云枢）会将这些模拟输入识别为真实的用户活动

**可行性评估**：✅ **可行**

`pynput` 发送的是系统级键盘事件，与真实键盘输入几乎无法区分，监控软件通常会将其识别为正常的工作活动。

**注意事项**：
1. 运行模拟输入时，确保有一个接收输入的窗口（如记事本）处于激活状态
2. 输入速度已设置为每字符150ms，模拟正常打字速度
3. 如需更真实的效果，可以在代码中添加随机延迟

## 原始需求文档

### 界面设计需求

* 应用名：QA任务进度提醒工具
* 应用风格：默认即可

* 左边设置一个输入框
    * 下方设置一按钮，显示开始

* 右边设置一个圆形时钟，显示当前的时间
    * 下方设置一个输出框
    * 设置按钮显示是否输出

* 界面大小初始化按照分辨率创建一个比较小的界面即可，用户可以自定义调整


### 功能实现需求

* 左边输入框功能为选择浏览本地文件 
* 左边按钮功能为启动或者停止信息推送操作
    * 点击启动后，定时调用webhook脚本进行信息推送,推送逻辑已写好，在根目录下的feishu_push.py

* 右边输入框功能为输出下一条即将推送的文本，需要逐字显示输出，一直循环
    * 此处希望可以模拟键盘输入的方式(可以让云枢监控到为工作时间即可)
* 右边按钮功能为启动或者停止字符输出
